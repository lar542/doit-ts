# 01-3. 타입스크립트 개발 환경 만들기

타입스크립트 개발 환경은 nodejs 개발 환경과 똑같다. 즉, nodejs를 설치하고 VSCode와 크롬 브라우저를 설치하면 바로 개발 가능하다. <br>

다음은 타입스크립트 컴파일러를 설치하기 위해 typescript 패키지를 설치하는 명령어 이다.<br>
타입스크립트는 nodejs 환경에서만 동작하기 때문에 nodejs의 패키지 매니저인 `npm`을 사용해서 설치한다.

typscript 패키지는 서버와 클라이언트로 동작하는 2개의 프로그램을 포함하고 있다. 따라서 타입스크립트 컴파일러 이름은 `tsc`이며, typescript compiler와 client라는 의미가 동시에 있다.
```hash
npm i -g typescript
tsc -v
```

다음은 ts 파일이 tsc에 의해 트랜스파일되어 js 파일을 생성하고, nodejs로 js 파일을 실행하는 명령어 이다.
```hash
tsc hello.ts
node hello.js
```

위처럼 tsc는 ts 코드를 ES5 형식의 js 코드로 변환만 할 뿐 실행하지는 않는데, 만약 변환과 실행을 동시에 하려면 ts-node라는 프로그램을 설치하면 된다.
```hash
npm i -g ts-node
ts-node hello.ts
```
