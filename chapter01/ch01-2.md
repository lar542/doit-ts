# 01-2. 타입스크립트 주요 문법 살펴보기

타입스크립트는 ESNext 문법을 대부분 지원하므로 타입스크립트를 다루려면 ESNext 문법을 알아야 한다.<br>
그리고 타입스크립트에만 고유한 문법도 있다. 이를 구분해서 알아보자.

## ESNext 주요 문법

### 비구조화 할당
비구조화 할당은 객체와 배열에 적용할 수 있다.<br>
👉 [03장에서 자세히 보기](../chapter03/ch03-4.md)

```javascript
// 객체의 속성을 얻는 예시
let person = { name: 'Jane', age: 22 }
let { name, age } = person // name : 'Jane', age : 23

// 배열의 첫 요소와 나머지 요소를 얻는 예시
let array = [1, 2, 3, 4]
let [head, ...rest] = array // head = 1, rest = [2, 3, 4]

// 값을 서로 교환(swap)하는 예시
let a = 1, b = 2
[a, b] = [b, a] // a = 2, b = 1
```

### 화살표 함수
👉 [04장에서 자세히 보기]
```javascript
// 자바스크립트에서 함수 선언
function add(a, b) { return a + b }
```

### 클래스
ESNext에서는 클래스를 제공해 객체지향 프로그래밍을 지원한다.<br>
ES5 자바스크립트로는 이런 형태의 코드를 작성하지 못한다.<br>
👉 [03장에서 자세히 보기](../chapter03/ch03-3.md)
```javascript
abstract class Animal {
  constructor(public name?: string, public age?: number) {}
  abstract say(): string
}

class Cat extends Animal {
    say() { return '야옹' }
}

class Dog extends Animal {
    say() { return '멍멍' }
}

let animals: Animal[] = [new Cat('야옹이', 2), new Dog('멍멍이', 3)]
let sounds = animals.map((a) => a.say()) // ['야옹', '멍멍']
```

### 생성기
- 타입스크립트 뿐만 아니라 파이썬이나 PHP 같은 몇몇 프로그래밍 언어는 `yield` 라는 키워드를 제공한다.
- yield 문 : 반복자를 의미하는 반복기(iterator)를 생성할 때 사용한다.
- 반복기는 반복기 제공자(iterable)를 통해 얻는다.
- yield 문을 이용해 반복기를 만들어 내는 반복기 제공자를 생성기(generator)라고 한다.
- generator는 `function*`과 `yield` 키워드를 이용해 만들며, 타입스크립트에서 `yield`는 반드시 `function*` 함수 내부에서만 사용할 수 있다.

👉 [06장에서 자세히 보기]
```javascript
function* gen() { // generator
    yield* [1, 2]
}
for (let value of get()) {
    console.log(value)
}
```

### Promise와 asnyc/await 구문
- ES5로 비동기 콜백 함수를 구현하면 코드가 복잡해 진다. = 콜백 지옥
- Promise는 웹 브라우저와 nodejs에서 제공되는 기본 타입으로 이 콜백 지옥을 해소하기 위해 만들어졌다.
- ESNext에서는 async/await 구문으로 여러 개의 Promise 호출을 더 간결하게 구현할 수 있게 한다.

👉 [07장에서 자세히 보기]
```javascript
async function get() { // Promise 형태로 반환
  let values = []
  // await는 Promise 객체를 해소(resolve)
  values.push(await Promise.resolve(1))
  values.push(await Promise.resolve(2))
  values.push(await Promise.resolve(3))
  return values
}
get().then(value => console.log(values)) // [1, 2, 3]
```

## 타입스크립트 고유 문법

### [타입 주석](../chapter03/ch03-1.md#타입-주석-type-annotation)과 [타입 추론](../chapter03/ch03-1.md#타입-추론-type-inference)
- 타입스크립트는 타입 주석(type annotation)을 생략할 수 있으며, 대입 연산자(=)의 오른쪽 값을 분석해 왼쪽 변수의 타입을 결정하는 타입 추론(type inference)이 가능하다.
- 타입스크립트의 타입 추론 기능은 자바스크립트의 소스코드와 호환성을 보장하는 데, 이 덕분에 자바스크립트로 작성된 `.js` 파일 확장자만 `.ts`로 바꿔도 타입스크립트 환경에서 바로 동작한다.
```javascript
let n: number = 1   // 타입 주석(type annotation)
let m = 2
```

### 튜플
튜플은 물리적으로는 배열과 같다.<br>
다만, 배열에 저장되는 아이템의 데이터 타입이 모두 같으면 배열, 다르면 튜플이라고 한다.<br>
👉 [05장에서 자세히 보기]
```javascript
let numberArray: number[] = [1, 2, 3] // 배열
let tuple: [boolean, number, string] = [true, 1, 'Ok'] // 튜플
```

### 대수 타입
대수 타입(algebraic data type, ADT) : 다른 자료형의 값을 가지는 자료형
- 합집합 타입(union 또는 sum type) : `|` 기호 사용
- 교집합 타입(intersection 또는 product tpye) : `&` 기호 사용

해당 기호를 사용하여 다음 예시처럼 여러 타입을 결합해서 만들 수 있다.<br>
👉 [10장에서 자세히 보기]
```javascript
type NumberOrString = number | string   // 합집합 타입
type AnimalAndPerson = Animal & Person  // 교집합 타입
```
