# 03-5. 객체의 타입 변환

## 타입 변환 (type conversion)
- 특정 타입의 변수 값을 다른 타입의 값으로 변환할 수 있는 기능 = 타입 변환(type conversion)
- type conversion은 다음 2가지 의미를 포함한다.
    - type casting : 명시적 타입 변환으로 코드에서 직접 표현하는 것
    - type coercion : 암시적 타입 변환으로 표현하지 않아도 컴파일러나 해석기가 알아서 처리하는 것

## 타입 단언(type assertion)
- 그런데 타입스크립트는 타입 변환이 아닌 타입 단언(type assertion)이라는 용어를 사용한다. 타입 단언문은 다음 2가지 형태가 있다.
```ts
(<타입>객체)
(객체 as 타입)
```

- 이들은 ES5 자바스크립트 구문이 아니기 때문에 자바스크립트의 타입 변환 구문과 구분하기 위해 타입 단언이라는 용어를 사용한다.
- 다음 타입 단언은 형태만 다를 뿐 내용상으로는 같다.
```ts
interface INameable {
    name: string
}

let obj: object = { name :'Jack' }

let name1 = (<INameable> obj).name
let name2 = (obj as INameable).name
```
